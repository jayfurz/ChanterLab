# ChanterLab Symbolic Chant Scripts

These files are seed fixtures for the proposed English-first symbolic script.
They are not authoritative hymn transcriptions.

Use them to test:

- symbolic timing,
- relative movement,
- rests,
- scale changes,
- checkpoint validation,
- lyric starts and melisma continuation,
- generated glyph display.
- gorgon, digorgon, and trigorgon temporal rewrite windows.
- repeated pthora and ison changes, including explicit ison octave/register.

The first parser should accept the strict grammar described in
`docs/BYZANTINE_CHANT_SCORE_PROPOSAL.md`.

When adding real practice hymns, include header comments with:

- source edition or manuscript,
- reviewer,
- copyright/public-domain status,
- mode/scale assumptions,
- any unresolved notation questions.
