# EXPAND-06: Score-Informed Multipart Assessment Research

Status: deferred research. Priority: P3.

Dependencies: stable single-singer scoring, audio benchmarks, trusted score data.

Owned files: isolated DSP/research harness and design evidence; no production UI
until go/no-go approval.

## Goal

Test whether one rehearsal-room device can estimate which expected choir section
is absent, flat, sharp, or masked using score-informed multipitch analysis.

## Method

Use the known expected pitches/times, synthetic mixtures, controlled ensemble
recordings, rooms/devices, part balance, vibrato, octave doubling, accompaniment,
and missing/incorrect sections. Measure false accusation rate first, then recall,
intonation error, latency, CPU/battery, and masking limits.

## Go/No-Go Gate

Production work requires robust real-recording evidence, near-zero harmful false
accusations, explainable insufficient-evidence outcomes, device feasibility, and
owner approval. Research success does not authorize recording/upload storage.
